3 · Attention heads — grouped-query attention

The single q/k/v shown above is really 32 parallel heads, each a 128-d slice of the projection matrices. Qwen3 groups them: every 4 query heads share one k/v head, shrinking the KV cache 4×.

canonical form
headi = Attention(x̂WQ(i), X̂WK(g(i)), X̂WV(g(i))), g(i) = ⌈i/4⌉
MHA(x̂) = Concat(head₁ … head₃₂)·WO

4 · The FFN — the block’s true hidden layer

After attention routes information between tokens, the FFN processes each token alone — the classic weighted-products-through-a-nonlinearity hidden layer, holding ~⅔ of all parameters.

canonical form
FFN₂₀₁₇(x) = max(0, xW₁ + b₁)·W₂
FFN_SwiGLU(x) = ( SiLU(xWgate) ⊙ xWup )·Wdown   ← Qwen3
